# MMKV-XCFramework

This repository hosts the MMKV iOS & macOS framwork as an XCFramework based Swift Package.
It contains,
- MMKV-iOS
- MMKV-iOS-Simulator
- MMKV-macOS-Catalyst
- MMKV-watchOS
- MMKV-tvOS
- MMKV-tvOS-Simulator

Use this repository's Swift Package in Xcode 15+ and Swift version 5.10+
